JavaScript

¿Que es?
JavaScript es un lenguaje de programación que se utiliza principalmente para crear páginas web dinámicas entre otras cosas.

JavaScript es un lenguaje de programación interpretado, no es necesario compilar los programas para ejecutarlos.
Los programas escritos con JavaScript se pueden probar directamente en cualquier navegador sin la necesidad de procesos intermedios.

¿Cuál es su función?
Curiosidad:
